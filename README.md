# Proste przetwarzanie obrazów w Pythonie

Celem projektu jest zaprezentowanie podstawowych operacji przetwarzania obrazów
z wykorzystaniem języka Python.

Program:
- wczytuje obraz rastrowy zdalnie przez link URL (urllib)
- wyświetla obraz z użyciem biblioteki matplotlib
- zmniejsza rozdzielczość obrazu o 50%
- konwertuje obraz do skali szarości (grayscale)
- obraca obraz o 90 stopni
- wyświetla obraz wynikowy
- prezentuje obraz w postaci macierzy liczb

## Wykorzystane biblioteki
- Python
- Pillow
- NumPy
- Matplotlib
